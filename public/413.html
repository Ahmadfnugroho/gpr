Error 413 - Ukuran File Terlalu Besar

File yang Anda upload melebihi batas ukuran maksimal yang diizinkan (1MB).

Solusi:

  • Kompres ukuran file Anda menjadi kurang dari 1MB
  • Gunakan fitur kompresi otomatis di halaman registrasi
  • Pilih file dengan ukuran yang lebih kecil
